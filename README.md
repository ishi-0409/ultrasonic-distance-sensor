# 超音波センサーによる距離測定装置（センチメートル表示版）

##  概要

このプロジェクトは、超音波センサー（HC-SR04）を用いて対象物との距離を**エコロケーション**の原理に基づいて測定する装置です。  
もともとインチ単位で表示していたものを、より直感的な**センチメートル単位で表示**するよう改良しました。
